## 2. Project Overview
**RS Sejahtera Medika Billing System**

Project ini merupakan implementasi **Database Management System (DBMS)** yang dibuat untuk membantu pengelolaan data pasien dan sistem billing di RS Sejahtera Medika. Sistem ini bertujuan untuk menggantikan proses pencatatan manual menjadi sistem berbasis database yang lebih terstruktur, efisien, dan meminimalisir kesalahan pencatatan.

Sistem ini diharapkan dapat mencatat data pasien, kunjungan, layanan medis, serta menghasilkan informasi tagihan secara lebih terorganisir.
---

## 3. Progress Project

Beberapa tahapan yang telah dikerjakan dalam project ini antara lain:
- Melakukan **analisis studi kasus** terhadap sistem billing rumah sakit
- Merancang **struktur database** dan relasi antar tabel
- Membuat **Entity Relationship Diagram (ERD)**
- Membuat **dataset tabel menggunakan Excel**
- Mengembangkan **aplikasi web untuk sistem billing**

Fitur web yang sudah tersedia:
- Dashboard
- Manajemen Pasien
- Kunjungan
- Billing
- Laporan

Saat ini sistem masih dalam tahap pengembangan pada bagian:
- Profile pengguna
- Penyempurnaan modul pasien
